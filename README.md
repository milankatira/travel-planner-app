
# Travel planner web 

Travel planner app allow you to create your pin(review) on map
all over the world and also you can read review
## Features

- Registration,Login and Logout
- write your review
- read other user's review
## Tech Stack

**Frontend:** React

**Backend:** Node, Express

**Database:** MongoDB
